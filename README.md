# 成都科瑞时代科技有限公司官网

## 🚀 项目简介

成都科瑞时代科技有限公司官方网站，采用现代化前端技术栈构建，具有炫酷的彗星粒子动态背景效果。

## ✨ 特色功能

### 彗星粒子背景
- 🌟 流星般的彗星效果，带有拖尾轨迹
- ⭐ 闪烁的星空背景
- 🖱️ 鼠标交互效果
- 🔗 科技感网格连接线
- 📱 响应式设计，支持高DPI屏幕

### 页面设计
- 深色科技感主题
- 渐变色文字和按钮
- 玻璃拟态卡片效果
- 流畅的悬停动画

## 🛠️ 技术栈

- **框架**: Next.js 14 (App Router)
- **语言**: TypeScript
- **样式**: Tailwind CSS
- **图标**: Lucide React
- **动画**: HTML5 Canvas + requestAnimationFrame

## 📦 快速开始

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build

# 启动生产服务器
npm start
```
